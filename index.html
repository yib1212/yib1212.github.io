Project and Research Experiences

  • Football League Schedule Optimization for The Premier League

    2019.07 - 2019.10

    The project focuses on the English Premier League (EPL) and optimizes the league schedule on the basis of the travelling tournament problem (TTP). We applied this method on the 2019/2020 season of EPL, and analyzed the advantages of the optimized schedule.

  • Design of the Control and Data Transmission System for FPGA-based Biped Robot

    2019.07 – 2020.06

    The course project designs the control system of the six-joint biped robot based on FPGA. The system uses Bluetooth to transfer user commands and real-time speed between the robot and smart phone and uses the data collected by the pressure sensor and the photoelectric encoder to design the control strategy for driving drive the robot.

  • Real-time 2D Human Pose Estimation Using Convolutional Neural Networks

    2019.07 - 2019.10

    The course project develops a method for real-time 2D human pose estimation combining heatmap regression and stacked hourglass convolutional neural networks. Given that hourglass-shaped CNN architectures are used de facto in semantic segmentation, object detection and pose estimation, our main contribution is re-designing the layers and obtaining a new hourglass network using only convolution and transposed convolution layers.

  • Design of the Smart Micro-grid System in Canberra, Australia

    2019.01 - 2019.06

    The team project applied system engineering to design a smart micro-grid system in Canberra, Australia, in which I was responsible for the analysis of system functions and function requirements.

  • Analysis of LiDAR-based Gully Erosion Mapping Data in North Queensland Using Machine Learning Models

    2019.01 - 2019.06

    The course project used a variety of machine learning models to analyze the gully erosion mapping data in Northern Queensland, Australia, compared the classification abilities of different models and marked the extent of gully erosion in this area, which provided reference data for the gully repair work.

  • Modelling and Analysis of Forest Fire Spread in Australian Capital Territory

    2018.07 - 2018.12

    The course project established a model of forest fire spread in the Australian Capital Territory based on Python, assessing the probability of fire at each location in the area according to the vegetation type, weather condition, humidity, etc., and marking the high-risk areas.

  • Research on the Upper-and-lower Computers Controlling Technology of LiDAR System

    2017.12 - 2018.06

    In this research project, the controlling module of the lower computer was designed based on FPGA, and the connection between the upper-and-lower computers was established through the configuration of the USB firmware, which synchronously transferred the system data including controlling commands and targets’ point cloud information.

  • Research on the Comparative Techniques of Calligraphy Original and Replica Based on OpenCV

    2017.03 - 2017.10

    The course project applied digital image processing technology to develop related programs based on C++ and its OpenCV library, which was used to find the small differences between Chinese calligraphy practitioners’ replica works and ancient original works. It analyzed the writing characteristics of practitioners and gave them suggestions for improvement.
